# Gmail Compose Function - Manual Testing Assignment

## Overview

This repository contains the manual testing assignment for the Test Craftsperson - Manual role at Incubyte.

The objective of this assignment is to test Gmail's Compose Email functionality for sending an email with:

* Subject: **Incubyte**
* Body: **QA test for Incubyte**

## Contents

### 1. Traditional Test Cases

A comprehensive set of positive and negative test cases covering:

* Email composition
* Recipient validation
* Subject and body validation
* Draft management
* Attachments
* CC/BCC functionality
* Error handling
* Usability and accessibility scenarios

### 2. BDD Test Cases

Behavior-Driven Development (BDD) scenarios written using the Given-When-Then format to validate Gmail Compose functionality from an end-user perspective.

## Test Coverage

The test suite covers:

* Functional Testing
* Validation Testing
* Negative Testing
* Usability Testing
* Accessibility Testing
* Compatibility Testing
* Error Handling Scenarios

* Traditional Style Test Cases (Excel)
